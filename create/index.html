How to create an Ad

To create an advertisement on Empties Ads you need to follow these quick steps.

1. Join the "Empties" Discord server

The first thing you must do is join the Empties Discord server. To do this, create a Discord account if you didn't already and verify your email. Then join the server and follow the guide (you will be asked to accept the rules and a few other things).

2. Create a request for a new Advertisement

Now that you joined the Discord server, locate the "Ads" section, go to #requests and create a new post. Follow the guide and write the answer to the questions before posting.
You may need to wait a maximum of 5 days before the request gets approved. Learn more about priorities.

3. How will I know when my request gets approved?

You will be notified in the channel #approved in the "Ads" section. You will get the link for the ad if you didn't specify to keep it private. If you specified it the same process will be done but a private thread will be created on the message you would receive. Click on it and you will find the Ad link.
